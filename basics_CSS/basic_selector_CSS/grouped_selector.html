Сгруппированные селекторы

Это хорошая возможность показать, как удобнее сокращать правила стилей. Если вам когда-нибудь понадобится применить одно и то же свойство к нескольким элементам, можете сгруппировать селекторы в одно правило, разделив их запятыми.

p, ul, a { color: navy; }

С помощью такого правила свойство CSS будет применено сразу к нескольким элементам (р, ul, a). Подобное группирование делает будущий процесс редактирования более эффективным и сокращает размер файла.

Отрицательной стороной такого способа выбора элементов согласно селекторам, конечно же, является то, что свойство (в данном случае, темно-синий цвет текста) будет применяться к каждому абзацу и другим перечисленным элементам в документе. А иногда требуется применить правило к конкретному абзацу или абзацам.
