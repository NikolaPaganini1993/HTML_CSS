Основные селекторы в CSS

Давайте рассмотрим на примерах один из самых важнейших механизмов CSS — селекторы. До сих пор мы использовали имена элементов в качестве селекторов. Вы видели, как селекторы могут быть сгруппированы вместе в разделенный запятыми список, так что допустимо применять свойства к нескольким элементам одновременно. Ниже приведен пример селектора элемента, который вы уже знаете.

p { color: navy; }

Кроме селектора элемента в СSS есть еще несколько основных селекторов, которые применяются на практике.

Примечание. Профессиональные веб-разработчики вместо селектора элемента используют селектор класса, о котором узнаете далее.
