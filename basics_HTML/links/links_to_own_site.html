Ссылки на страницы собственного сайта

Наиболее простым относительным URL-адресом является ссылка на дру­гой файл в пределах одного каталога. При создании ссылки на файл в том же каталоге нужно указать только имя файла (с расширением). Когда URL-адрес представляет собой лишь имя файла, сервер ищет его в теку­щем каталоге (то есть в том, где находится HTML-документ со ссылкой). В этом примере сделаем ссылку с главной страницы (index.html) на страницу с информацией о сайте (titles.html). Оба файла находятся в одном корневом каталоге.
