Ссылки

Всемирную паутину создал тег a. Он добавляет ссылки, превращая текст веб-страницы в гипертекст, то есть «больше, чем текст».

Для дополнительных сведений у тегов есть атрибуты. Так, адрес хранится в атрибуте с именем href (от англ. hypertext reference, «гипертекстовая ссылка»). Имя атрибута говорит браузеру, какой признак нужно поменять, а значение — каким этот признак должен стать. Значение атрибута href представляет собой адрес ресурса, на который можно перейти. Как устроена гиперссылка, ведущая на сайт ya.ru

Для создания ссылки нам потребуется следующий код:

<a href="https://ya.ru/" target="_blank">YANDEX.RU>
